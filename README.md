
# Amazon Product Web Scraping Project

This project is focused on **Web Scraping Amazon Products** using Python with the help of the `Selenium` and `BeautifulSoup` libraries. It automates the process of opening dynamic web pages, extracts product information, and stores the data for analysis.

## Overview

- **Selenium**: Used to interact with and open dynamic web pages in a Chrome browser using a web driver.
- **BeautifulSoup**: Used for parsing HTML and extracting the required information present in specific HTML tags.

## Features

- Scrape product titles, prices, ratings, and reviews from Amazon product pages.
- Ability to scrape multiple pages using dynamic page navigation.
- Automated browser interaction using Selenium WebDriver.
- Extract structured data from the HTML content using BeautifulSoup.
- Save extracted data into a CSV file for further analysis.
